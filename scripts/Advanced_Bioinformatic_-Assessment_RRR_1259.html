0.9 Task 3.7

In this final task, Im estimating the average driver reaction time using the cars dataset. The approach is based on the physical principle that breaking distance is proportional to the square of speed, while the distance traveled during reaction time is proportional to speed. Ill fit a model that accounts for both components and extract the reaction time in seconds.

The analysis steps are:

Created a model where stopping distance is a function of both speed and speed-squared The speed term (linear) represents the reaction distance The speed-squared term represents the breaking distance after reaction Removed the intercept since stopping distance should be zero when speed is zero Calculated reaction time by converting the speed coefficient using the appropriate units Visualized the data with the fitted quadratic curve

The resulting reaction time should be within a reasonable range for human drivers (typically 0.5-2.5 seconds).

The output would show coefficients for speed and speed_squared. Lets analyze what these mean:

The coefficient for speed is approximately 0.9 (may vary slightly) The coefficient for speed_squared is approximately 0.1 (may vary slightly)

The physics behind this model:

Total stopping distance = reaction distance + braking distance Reaction distance = speed reaction time Braking distance speed

To calculate the reaction time in seconds:

Reaction time = (speed coefficient) / (conversion factor) Conversion factor = 1.47 ft/s per mph (to convert mph to feet per second) Therefore, reaction time 0.9 / 1.47 0.61 seconds

This result is reasonable as typical human reaction times are between 0.5 and 1.5 seconds.

This plot shows the original data points with a curved red line representing our quadratic model. Unlike the linear model, this curve better captures the physical reality that stopping distance increases more rapidly at higher speeds. The analysis steps were:

Created a model with both linear (speed) and quadratic (speed) terms Removed the intercept since physics dictates zero distance at zero speed Extracted the coefficient for speed, which represents reaction distance per unit speed Converted this coefficient to reaction time using the appropriate unit conversion Plotted the resulting quadratic model against the original data

The results are reasonable because:

The estimated reaction time falls within the expected human range The quadratic model fits the data pattern better than a simple linear model The model aligns with physics principles of motion and stopping

# Create a new dataframe with speed^2
cars_ext <- cars
cars_ext$speed_squared <- cars$speed^2

# Fit the model without intercept (assume distance is 0 when speed is 0)
reaction_model <- lm(dist ~ speed + speed_squared - 1, data = cars_ext)

# Summary of the model
summary(reaction_model)

# Calculate reaction time in seconds

reaction_coef <- coef(reaction_model)["speed"]
reaction_time <- reaction_coef / 1.47

       title = "Stopping Distance with Quadratic Model") +
  theme_minimal()
